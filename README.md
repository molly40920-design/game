## 🚀 部署方式

### GitHub Pages
1. 建立新 Repository
2. 上傳 `index.html`
3. Settings → Pages → Deploy from branch → main → Save
4. 約 1 分鐘後上線：`https://帳號.github.io/倉庫名/`

### Antigravity.ai
1. 登入 https://antigravity.ai
2. 建立新 Project
3. 上傳 `index.html` 作為網站檔案
4. 設定自訂網域（選填）

### 任意靜態託管
直接上傳 `index.html` 即可，無需伺服器、無需建構工具。

## 🏗️ 技術架構

- **純前端**：HTML5 Canvas + Vanilla JS，零外部依賴
- **字體**：Google Fonts（Orbitron + Noto Sans TC + Press Start 2P）
- **音效**：Web Audio API 即時合成（20 種音效）
- **RWD**：支援桌面/平板/手機
- **單檔**：所有 CSS + JS + 200 款遊戲打包在一個 HTML 檔
